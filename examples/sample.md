# 第一部分 公司介绍

## 一、公司简介

示例科技是一家专注于汽车行业信息化的高新技术企业，成立于 2010 年，总部位于北京。公司拥有员工 500 余人，其中研发人员占比超过 60%。

## 二、我们的优势

- **专业性强**：深耕汽车行业 10 余年
- **产品领先**：自主研发的 ERP/MES/QMS 等系列产品
- **服务优质**：7×24 小时全国服务网络
- **团队专业**：汇聚行业顶级专家

# 第三部分 技术方案

## 一、项目背景

随着汽车行业向智能化、电动化转型，质量管理的复杂度急剧上升……

## 二、技术架构

> 系统采用微服务架构，前后端分离，云原生部署。

### 2.1 前端技术栈

- Vue 3.x
- TypeScript
- Element Plus

### 2.2 后端技术栈

- Spring Cloud Alibaba
- Nacos / Sentinel / Seata
- MySQL 8.0 集群
